# FAST-AR: Framework for Accelerated Spatial awareness in Augmented Reality

Designed to provide accelerated gesture recognition through hardware acceleration of image preprocessing combined with the Xilinx DPU to enable high speed recognition.
Additionally, a Rubik Pi provides video passthrough and "pseudo-stereo" display for the user, allowing for easier integration of the AUP-ZU3 provided gesture data compared to a PL only approach

Note that some work is not present in this repository to prevent bloat. For example PCAM5 video and DPU are in separate repositories or only pieces of them are stored here respectively.
The PCAM repository is separate from this one and can be found at https://github.com/Henrywwg/pcam5.

Uses RGB888 color version
